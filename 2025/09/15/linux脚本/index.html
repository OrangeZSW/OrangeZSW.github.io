Linux 脚本（Bash）基础教程

1. 什么是 Linux 脚本？

  • Linux 脚本是用来自动化执行一系列命令的文本文件。
  • 常用的脚本语言是 Bash（Bourne Again SHell）。
  • 脚本通常以 .sh 结尾，但不强制。

2. 第一个脚本：Hello World

  • #!/bin/bash ：告诉系统用 bash 来执行脚本
  • echo ：打印输出

3. 变量和基本运算

  • 变量赋值不能有空格。
  • 使用 $(( )) 进行算术运算。
  • 变量引用时前面加 $。

8. 脚本参数

脚本执行时可以带参数：

在脚本里：

  • $0 脚本名
  • $1 第一个参数
  • $2 第二个参数
  • $# 参数个数
  • $@ 所有参数

9. 常用命令示例

  • ls 列目录
  • cd 切目录
  • cat 显示文件内容
  • grep 查找内容
  • awk 处理文本
  • sed 文本替换
