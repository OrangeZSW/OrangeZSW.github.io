⚙️配置

​ 1️⃣登录必应网站管理

​ 2️⃣选择从 GSC 导入

​ 3️⃣ 将下载到的 xml 文件放入网站资源文件夹的根目录下，比如我的是 Hexo 的 source 文件夹下。
